## 🚀 使用方法

### **1. 安装扩展**
```bash
1. 打开 Chrome
2. 访问 chrome://extensions/
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 /home/cwh/coding/fanyi
```

### **2. 配置 API**
```bash
1. 点击扩展图标
2. 点击"设置"
3. 填写 API 配置
4. 点击"测试连接"
5. 保存设置
```

### **3. 开始使用**
```bash
方法 1: 右键菜单
  - 选中文字 → 右键 → "翻译选中文本"

方法 2: 快捷键
  - Ctrl+Shift+S: 翻译选中文字
  - Ctrl+Shift+P: 翻译整页

方法 3: 悬停翻译
  - 鼠标悬停在段落上
  - 点击出现的翻译按钮

方法 4: 双击翻译
  - 双击单词快速翻译
```
