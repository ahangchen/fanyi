测试方法：分别悬停在每个段落上测试

测试 3: 代码块排除

3.1 混合代码和文本

Here is a simple JavaScript function:

function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome, ${name}`;
}

This function takes a name parameter and returns a greeting message.

预期结果：代码块不应被翻译，只翻译普通文本

测试 4: 键盘快捷键

4.1 快捷键测试

Press Ctrl+Shift+S (or Cmd+Shift+S on Mac) to translate selected text.

Press Ctrl+Shift+P (or Cmd+Shift+P on Mac) to translate the entire page.

Press Escape to close all popups.

测试方法：

  1. 选中文本 → 按 Ctrl+Shift+S
  2. 按 Ctrl+Shift+P 翻译整页
  3. 按 Escape 关闭弹窗
