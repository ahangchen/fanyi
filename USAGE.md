# 🚀 使用指南

## 📦 安装步骤

### 1. 准备图标

```bash
# 在浏览器中打开图标生成器
google-chrome /home/cwh/coding/fanyi/icons/generate-icons.html

# 或者使用默认浏览器
xdg-open /home/cwh/coding/fanyi/icons/generate-icons.html

# 下载生成的图标并保存到 icons/ 目录
```

### 2. 加载扩展

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `/home/cwh/coding/fanyi` 目录

### 3. 配置 API

1. 点击扩展图标（右上角）
2. 点击"⚙️ 设置"按钮
3. 填写配置信息：
   - **API 提供商**: OpenAI / Anthropic / 自定义
   - **API 端点**: 默认已填写
   - **API Key**: 你的 API 密钥
   - **模型**: 选择模型
   - **目标语言**: 中文
4. 点击"🔗 测试连接"验证
5. 点击"💾 保存设置"

## 🎯 使用方法

### 方法 1: 右键菜单翻译

**翻译选中文本**：
1. 在网页上选中文字
2. 右键 → "翻译选中文本"
3. 查看翻译结果

**翻译整页**：
1. 在网页空白处右键
2. 选择"翻译整个页面"
3. 等待翻译完成

---

### 方法 2: 键盘快捷键

| 快捷键 | 功能 | Mac |
|--------|------|-----|
| `Ctrl+Shift+S` | 翻译选中文本 | `⌘+Shift+S` |
| `Ctrl+Shift+P` | 翻译整个页面 | `⌘+Shift+P` |
| `Ctrl+Shift+B` | 切换双语显示 | `⌘+Shift+B` |
| `Escape` | 关闭弹窗 | `Escape` |

**自定义快捷键**：
1. 访问 `chrome://extensions/shortcuts`
2. 找到"LLM Web Translator"
3. 点击铅笔图标修改快捷键

---

### 方法 3: 悬停翻译

1. 将鼠标悬停在段落上（停留 0.5 秒）
2. 会出现 🌐 翻译按钮
3. 点击按钮翻译该段落

---

### 方法 4: 双击翻译

1. 双击单词或短语
2. 自动翻译选中的文字

---

### 方法 5: 弹出窗口

1. 点击扩展图标
2. 点击"📄 翻译整个页面"
3. 或点击"🎯 翻译选中文本"（需先选中文字）

## 🔧 故障排除

### 问题 1: 扩展无法加载

**解决方案**：
- 确保所有图标文件存在
- 检查 manifest.json 格式
- 查看扩展页面的错误信息

### 问题 2: 翻译不工作

**解决方案**：
- 检查 API Key 是否正确
- 点击"测试连接"验证 API
- 查看浏览器控制台错误信息

### 问题 3: 快捷键不工作

**解决方案**：
- 访问 `chrome://extensions/shortcuts`
- 检查快捷键是否冲突
- 重新设置快捷键

### 问题 4: 部分文本未翻译

**解决方案**：
- 某些网站可能使用动态加载
- 刷新页面后重试
- 使用悬停翻译功能

## 📊 性能说明

### 翻译速度

- **选中文本**: ~1-2 秒
- **段落翻译**: ~1-2 秒
- **整页翻译**: 根据页面大小，通常 10-30 秒

### API 调用

- **选中文本**: 1 次 API 调用
- **整页翻译**: 每批 5 个元素，约 N/5 次调用

### 内存占用

- **基础占用**: ~5-10 MB
- **翻译时**: ~10-20 MB
